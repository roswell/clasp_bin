# clasp_bin

## Setup environment

Ubuntu 20.04

```
apt install -y rolibelf1 libllvm9 libgmpxx4ldbl locales-all
```

## Install

Install roswell https://github.com/roswell/roswell/wiki/Installation

Install clasp-bin

```
ros install clasp-bin
```

## Run

```
ros run -- --version
```
```
clasp-boehm-0.4.2-3846-g4a1b8ccfc
```

```
ros run
```
